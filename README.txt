# Name Sorter App

This C# program sorts a list of names first by last name and then by given names. It takes an unsorted list of names from a file, sorts them, and outputs the sorted list to the console and to a file.

## Usage

1. Clone the repository:


git clone

2. Navigate to the project directory

3. Build the solution:

dotnet build

4. Run the program with the path to the unsorted names file as an argument:

dotnet run "C:\Users\asilw\source\repos\D&Dassesmment\nameSorterApp\unsorted-names-list.txt"


5. To run the tests

dotnet test
